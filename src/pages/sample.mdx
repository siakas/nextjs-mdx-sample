# 表示／非表示の切り替えのための display: none

ある要素の表示と非表示の切り替えを CSS でおこなう際、 `display: block` と `display: none` とで切り替えの指定をすることがあります。

```scss
.obj {
  display: none;

  &.is-active {
    display: block;
  }
}
```

しかし `display: block` と `display: none` による切り替えはスタイルの指定が冗長であるだけでなく、 `display: block` に切り替えた要素に、元々は `display: flex` が適用されていた場合などでは `display` プロパティの値が変わってしまうことで、思わぬ表示崩れの原因になることもあります。

## 今はこう書ける

今は `:not()` を使うことでよりシンプルに、 `display: none` の指定のみで切り替えのスタイルを制御できます。

```scss
.obj {
  &:not(.is-active) {
    display: none;
  }
}
```
